# 🇪🇸 Español

## Descripción

WebM to MP3 Converter es una aplicación de escritorio desarrollada en Python que permite convertir archivos de audio o video en formato `.webm` a `.mp3` de forma rápida y sencilla.

La herramienta está pensada para usuarios que necesitan extraer audio de archivos WebM mediante una interfaz simple e intuitiva.

## Características

* Conversión de archivos `.webm` a `.mp3`.
* Soporte para múltiples archivos.
* Función de arrastrar y soltar (Drag & Drop).
* Selección manual de archivos.
* Conversión rápida mediante FFmpeg.
* Los archivos convertidos se guardan automáticamente en la misma carpeta que el original.
* Interfaz gráfica amigable.

## Tecnologías Utilizadas

* Python
* Tkinter / CustomTkinter
* FFmpeg
* File Handling

## Instalación

Clonar el repositorio:

```bash
git clone https://github.com/fyjoaco/webm-to-mp3-converter.git
cd webm-to-mp3-converter
```

Instalar dependencias:

```bash
pip install -r requirements.txt
```

Ejecutar:

```bash
python main.py
```

## Cómo Usar

1. Abrir la aplicación.
2. Arrastrar uno o varios archivos `.webm` a la ventana.
3. O presionar "Seleccionar archivos".
4. Esperar a que finalice la conversión.
5. Los archivos `.mp3` aparecerán en la misma carpeta que los originales.

# 🇺🇸 English

## Description

WebM to MP3 Converter is a Python desktop application designed to quickly and easily convert `.webm` audio or video files into `.mp3` format.

The tool provides a simple and user-friendly interface for extracting audio from WebM files.

## Features

* Convert `.webm` files to `.mp3`.
* Multiple file support.
* Drag & Drop functionality.
* Manual file selection.
* Fast conversion powered by FFmpeg.
* Converted files are automatically saved in the same folder as the source file.
* User-friendly graphical interface.

## Technologies Used

* Python
* Tkinter / CustomTkinter
* FFmpeg
* File Handling

## Installation

Clone the repository:

```bash
git clone https://github.com/fyjoaco/webm-to-mp3-converter.git
cd webm-to-mp3-converter
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run:

```bash
python main.py
```

## How to Use

1. Open the application.
2. Drag and drop one or more `.webm` files into the window.
3. Or click "Select Files".
4. Wait for the conversion to finish.
5. The generated `.mp3` files will be saved in the same directory as the original files.
